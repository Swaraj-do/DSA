# Problem 15 — Arrays DS

**Link:** https://www.hackerrank.com/challenges/arrays-ds/problem  
**Difficulty:** Easy  
**Platform:** HackerRank

---

## Approach

Reverse the array using slicing a[::-1].
a[::-1] means start from end, go to beginning, step -1.

## Dry run — a=[1,4,3,2]

a[::-1] → [2,3,4,1] ✓

## Complexity

- Time: O(n)
- Space: O(1)
