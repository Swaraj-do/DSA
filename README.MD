# DSA

Daily DSA practice — problems, solutions and notes.

---

## Folder structure
```
DSA/
├── problem/
│   ├── solution.py
│   └── notes.md
└── README.md
```

---

## How to run a solution
```bash
cd problem-1
python solution.py
```

Change `problem-1` to whichever problem folder you want to run.
